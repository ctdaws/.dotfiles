# Dotfiles managed using Nix

## Usage
- Install `nix` using the Determinate Systems installer
- Install `nix-darwin`
- Clone the repo to `~/.dotfiles` 
- Run the command `darwin-rebuild switch --flake ~/.dotfiles#personal`
